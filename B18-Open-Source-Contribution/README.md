## Contribution Summary

I improved the repository’s documentation to make it clearer and more accessible for beginners. This included fixing a broken PDF download link, adding a table of contents for navigation, introducing prerequisite and usage sections, and adding a cybersecurity context explaining how Rust helps reduce common vulnerabilities such as memory safety issues.
