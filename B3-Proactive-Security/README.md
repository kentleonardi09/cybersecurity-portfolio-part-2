# B3 – Discover 3 Proactive Security Implementations in Practice

This section presents three proactive security implementations identified through direct interaction with a live system environment. Each example demonstrates how security mechanisms actively protect systems by preventing threats before they occur.

## 1. Automatic Security Updates

### Description
Automatic security updates are a proactive security implementation that ensures operating systems receive the latest patches and vulnerability fixes without requiring manual intervention.

## Final Reflection

The three implementations identified—automatic updates, firewall protection, and antivirus software—demonstrate how proactive security measures operate in practice. These mechanisms continuously monitor, detect, and prevent threats, significantly reducing the likelihood of successful attacks. Their presence highlights the importance of maintaining active and layered security controls in modern computing environments.
