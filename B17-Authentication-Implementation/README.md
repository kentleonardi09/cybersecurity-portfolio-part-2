# B17 - Implement One State-of-the-Art Solution and Evaluate It

## Overview
This task demonstrates the implementation of a modern cybersecurity solution and evaluates its effectiveness in protecting user accounts in real-world environments.

---

## 1. Chosen Solution: Multi-Factor Authentication (MFA) using Google Account Security

### Description
For this task, I implemented a state-of-the-art authentication system by securing my Google account using Multi-Factor Authentication (MFA). This included enabling 2-Step Verification, which requires an additional verification step beyond just a password when logging in.

The system includes multiple authentication methods such as Google prompts, an authenticator app, and phone verification. These methods provide layered security and reduce the risk of unauthorized access. I also explored modern authentication features such as passkeys, which represent a newer approach to passwordless security.

---

### Implementation
I enabled 2-Step Verification on my Google account and configured multiple authentication options, including:
- Google Prompt for quick login approval
- Authenticator app for time-based verification codes
- Phone number verification as a backup method
- Backup codes for account recovery

Additionally, I used Google Password Manager to store and manage my login credentials securely, ensuring the use of strong and unique passwords.

---

### Evaluation
The implementation was straightforward and significantly improved account security. By requiring multiple verification steps, the system provides strong protection even if a password is compromised. The use of different authentication methods increases reliability and flexibility.

One major advantage is ease of use, as Google integrates these features seamlessly across devices. The inclusion of backup options also ensures account recovery in case of device loss.

However, a limitation is that users must manage their security settings carefully. If authentication devices or backup methods are not maintained, account recovery can become difficult. Additionally, while passkeys are a modern feature, they may not be available on all devices.

Overall, this solution demonstrates a practical and widely used approach to securing accounts in real-world systems.
